# Contributing to Qwik UI Fluffy

We welcome and appreciate contributions from the community!

Whether you're looking to submit bug fixes, improve documentation, or add new features, your efforts can
make a significant impact on the project. By contributing, you'll also be joining
our vibrant and supportive community of developers.

Our team will review your pull request and, if approved, merge it into the main repository.

[Read our contributing guidelines](https://github.com/qwikifiers/qwik-ui/blob/main/CONTRIBUTING.md)
