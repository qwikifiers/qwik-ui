## CSS Scroll snapping

```css
@layer qwik-ui {
  @media (pointer: coarse) {
    [data-qui-carousel-scroller][data-draggable] {
      scroll-snap-type: x mandatory;
      overflow-x: scroll;
    }

    [data-draggable] [data-qui-carousel-slide] {
      scroll-snap-align: start;
    }
  }
}
```
