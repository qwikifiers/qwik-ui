# Toggle

A two-state button that can be either on or off.

## Usage / Component State

### Initial Value (Uncontrolled)

An initial, uncontrolled value can be provided using the `defaultPressed` prop.

<Showcase name="defaultPressed" />

### Reactive Value (Controlled)

Pass the `pressed` prop to make the pressed state controlled.
The `onPressedChange$` event is fired when the user toggle the button, and receives the new value.

<Showcase name="pressed" />

### Disabled

Pass the `disabled` prop.

<Showcase name="disabled" />
