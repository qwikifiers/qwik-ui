# Toggle Group

A set of two-state buttons that can be toggled on or off.

## Usage / Component State

### Multiple selection

<Showcase name="multiple" />

### Initial Value (Uncontrolled)

An initial, uncontrolled value can be provided using the `defaultValue` prop.

<Showcase name="defaultValue" />

### Reactive Value (Controlled)

Pass the `value` prop to make the pressed state controlled.
The `onValueChange$` event is fired when the user toggle the button, and receives the new value.

<Showcase name="value" />

Or you can bind the value with a signal.

<Showcase name="value-bind" />

### Disabled

Pass the `disabled` prop.

<Showcase name="disabled" />

You can also disabled specific items, pass the `disabled` prop at the `ToggleGroup.Item` level:
When navigating with key arrows, the disabled item will be skipped.

<Showcase name="item-disabled-center" />

### Looping Enabled

Pass the `loop` prop. When enabled, keyboard navigation will loop from last item to first, and vice versa.
If one item is disabled it will skip it.

<Showcase name="loop" />

If one item is disabled it will skip it.

<Showcase name="loop-item-disabled" />

### Vertical Orientation

Pass the `orientation` prop.

<Showcase name="vertical" />

### Right-to-Left (rtl) Direction

Pass the `direction` prop to `rtl`.

<Showcase name="horizontal-rtl" />
