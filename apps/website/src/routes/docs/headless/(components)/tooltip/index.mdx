# Tooltip

#### A tooltip that displays information related to an element when the element receives keyboard focus or the mouse hovers over it.

<MainExample>
  ```tsx
  <Tooltip
    content="I'm a tooltip!"
    class="bg-slate-200 dark:bg-gray-900 p-4 rounded-xl"
  >
    Hover over me!
  </Tooltip>
  ```
</MainExample>

## Building blocks

  <CodeExample>
    ```tsx
    import { component$ } from '@builder.io/qwik';
    import { Tooltip } from '@qwik-ui/headless';

    export default component$(() => (
      <Tooltip content='Tooltip content'>
        Trigger content
      </Tooltip>
    ));
    ```

  </CodeExample>

## Examples

### EXAMPLE: Describing Icon with Tooltip

<Example1>
  ```tsx
  <Tooltip
    content="Clicking this icon takes you to Qwik UI's GitHub repository!"
    class="bg-slate-200 dark:bg-gray-900 p-4 rounded-xl"
  >
    <a href="https://github.com/qwikifiers/qwik-ui" class="mx-auto">
      <GitHubIcon />
    </a>
  </Tooltip>
  ```
</Example1>
