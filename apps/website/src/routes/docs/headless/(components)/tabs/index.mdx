## Building blocks

  <CodeExample>
    ```tsx
    import { component$ } from '@builder.io/qwik';
    import { Tab, TabList, TabPanel, Tabs } from '@qwik-ui/headless';

    export default component$(() => {

      return (
        <>
          <Tabs>
            <TabList>
              <Tab>Tab 1</Tab>
              <Tab>Tab 2</Tab>
              <Tab>Tab 3</Tab>
            </TabList>

            <TabPanel>Content 1</TabPanel>
            <TabPanel>Content 2</TabPanel>
            <TabPanel>Content 3</TabPanel>
          </Tabs>
        </>
      );
    });
    ```

  </CodeExample>

## Disabled

You can disable a tab by setting the `disabled` prop to true.

<DisabledTabsExample>
  ```tsx
  <Tabs>
    <h3>Dad jokes</h3>
    <TabList>
      <Tab disabled={true}>Disabled Tab</Tab>
      <Tab>Joke 2</Tab>
      <Tab>Joke 3</Tab>
      <Tab>Joke 4</Tab>
    </TabList>

    <TabPanel>"What did the fish say when he hit the wall", " Dam."</TabPanel>
  </Tabs>
  ```
</DisabledTabsExample>

## Behavior

### Automatic

The default behavior of the tabs is manual, which means that the tabs will automatically switch to the next tab when the user hovers over the tab. You can change this behavior by setting the `behavior` prop to `automatic`.

<AutomaticBehaviorTabsExample>
  ```tsx
  <Tabs behavior="automatic">
    <h3>Danish Composers</h3>
    <TabList>
      <Tab>Maria Ahlefeldt</Tab>
      <Tab>Carl Andersen</Tab>
      <Tab>Ida Henriette da Fonseca</Tab>
    </TabList>
    <TabPanel>
      <p>
        Maria Theresia Ahlefeldt (16 January 1755 - 20 December 1810) was a ...
      </p>
    </TabPanel>
    <TabPanel>
      <p>Carl Joachim Andersen (29 April 1847 - 7 May 1909) was a ...</p>
    </TabPanel>
    <TabPanel>
      <p>Ida Henriette da Fonseca (July 27, 1802 - July 6, 1858) was a ...</p>
    </TabPanel>
  </Tabs>
  ```
</AutomaticBehaviorTabsExample>

### Manual

You can set the behavior to manual, which means that the tabs will not automatically switch to the next tab when the user presses the right arrow key, and to the previous tab when the user presses the left arrow key.

<ManualBehaviorTabsExample>
  ```tsx
  <h3>Danish Composers</h3>
  <Tabs behavior="manual">
    <TabList>
      <Tab>Maria Ahlefeldt</Tab>

      <Tab>Ida Henriette da Fonseca</Tab>
    </TabList>
    <TabPanel>
      <p>

## Dynamic

<DynamicTabsExample>
```tsx

const tabsState = useStore([
  'Dynamic Tab 1',
  'Dynamic Tab 2',
  'Dynamic Tab 3',
]);

return <>
  
  <Tabs>
    <TabList>
      {tabsState.map((tab) => (
        <Tab key={tab}>{tab}</Tab>
      ))}
    </TabList>
    {tabsState.map((tab) => (
      <TabPanel key={tab}>{tab} Panel</TabPanel>
    ))}
  </Tabs>

  <button style={{ color: 'red', marginTop: '1rem' }}
    onClick$={() => {
      if (tabsState.length > 1) {
        tabsState.splice(0, 1)
      }
    }}>
    Remove First Tab
  </button>
</>
```
</DynamicTabsExample>

## onSelectedIndexChange$

You can listen to changes in the selected index by subscribing to the `onSelectedIndexChange$` store.

<OnSelectedIndexChangeTabsExample>
```tsx

const selectedIndexSig = useSignal(0);

return <div>

{' '}

<Tabs
  onSelectedIndexChange$={(index) => {
    selectedIndexSig.value = index;
  }}
>
  <h3>Danish Composers</h3>
  <TabList>
    <Tab>Maria Ahlefeldt</Tab>
    <Tab>Carl Andersen</Tab>
    <Tab>Ida Henriette da Fonseca</Tab>
  </TabList>
  <TabPanel>
    <p>
      Maria Theresia Ahlefeldt (16 January 1755 - 20 December 1810) was a ...
    </p>
  </TabPanel>
  <TabPanel>
    <p>Carl Joachim Andersen (29 April 1847 - 7 May 1909) was a ...</p>
  </TabPanel>
  <TabPanel>
    <p>Ida Henriette da Fonseca (July 27, 1802 - July 6, 1858) was a ...</p>
  </TabPanel>
</Tabs>

  <p>Selected Index: {selectedIndexSig.value}</p>
</div>
```
</OnSelectedIndexChangeTabsExample>

## onClick$

You can pass a custom `onClick$` handler.

<OnClickTabsExample>
```tsx
const tabsClickedCountSig = useSignal(0);

return <div>

<h3>Danish Composers</h3>
<h4>(watch the counter below)</h4>
<Tabs
  onClick$={() => {
    tabsClickedCountSig.value++;
  }}
>
  <TabList>
    <Tab>Maria Ahlefeldt</Tab>
    <Tab>Carl Andersen</Tab>
    <Tab>Ida Henriette da Fonseca</Tab>
  </TabList>
  <TabPanel>
    <p>
      Maria Theresia Ahlefeldt (16 January 1755 - 20 December 1810) was a ...
    </p>
  </TabPanel>
  <TabPanel>
    <p>Carl Joachim Andersen (29 April 1847 - 7 May 1909) was a ...</p>
  </TabPanel>
  <TabPanel>
    <p>Ida Henriette da Fonseca (July 27, 1802 - July 6, 1858) was a ...</p>
  </TabPanel>
</Tabs>

<p>Click count: {tabsClickedCountSig.value}</p>

</div>
```
</OnClickTabsExample>
