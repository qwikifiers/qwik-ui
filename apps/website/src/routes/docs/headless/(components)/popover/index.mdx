# Popover

A popover is an element that displays a modal with content when the user clicks or hovers on a trigger element.

<MainExample>
  ```tsx
  <Popover>
    <PopoverContent>
      <div class="bg-gray-500 p-4">Hi, I'm the content</div>
    </PopoverContent>
    <PopoverTrigger> Click on me </PopoverTrigger>
  </Popover>
  ```
</MainExample>

## Building blocks

```tsx
import { component$ } from '@builder.io/qwik';
import { Popover } from '@qwik-ui/headless';

export default component$(() => (
  <Popover>
    <PopoverContent>
      <div class="bg-gray-500 p-4">Hi, I'm the content</div>
    </PopoverContent>
    <PopoverTrigger> Click on me </PopoverTrigger>
  </Popover>
));
```

## Examples

### EXAMPLE: Positioning the popover's content (on top)

<Example1>
  ```tsx
  <Popover placement="top">
    <PopoverContent>
      <div class="bg-gray-500 p-4">Hi, I'm the content, but now on top</div>
    </PopoverContent>
    <PopoverTrigger> Click on me </PopoverTrigger>
  </Popover>
  ```
</Example1>
