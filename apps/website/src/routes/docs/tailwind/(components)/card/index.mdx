# Card

#### Cards are versatile and can be used in various contexts, such as displaying product information, showcasing user profiles or presenting news articles.

{' '}

## Basic Card

<Example01>
  ```tsx
  <Card>
    <CardBody>
      <p>
        Qwik is a new kind of web framework that can deliver instant loading web
        applications at any size or complexity. Your sites and apps can boot
        with about 1kb of JS.
      </p>
    </CardBody>
  </Card>
  ```
</Example01>
